############# README ############



Steps:

1. PC should have VREP and Python 2.7 installed.
2. Download the project folder
3. If not already directed, direct Python directory to project 
folder (usually python will do this on its own when you run 
the project_main.py file)
4. Open "Project_scene.ttt" scene in VREP simulator
5. Run simulation
6. Run "Project_main.py"

the robot will start moving!
